# Gizmo Store Print Shop

A 3D print marketplace starter website built with Vite and TypeScript. The site is designed for three user roles:

- **Designers** who upload CAD and 3D print files.
- **Printers** who manufacture and ship printed parts.
- **Buyers** who browse designs, place orders, and receive deliveries.

The model supports revenue sharing so designers earn royalties, printers earn production fees, and the platform earns commissions.

## Available scripts

- `npm install` — install dependencies
- `npm run dev` — start the local development server
- `npm run build` — build the production site
- `npm run preview` — preview the production build locally

## Next steps

1. Open `src/main.ts` to adjust the homepage content or add user flows.
2. Edit `src/style.css` to update branding, colors, and layout.
3. Add real marketplace data, authentication, and purchase workflows as needed.
